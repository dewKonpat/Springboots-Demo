## *Installation*
* Lombok plugin
    1. Find lombok Jar in C:\Users\xxxx\\.m2\repository\org\projectlombok\lombok\1.18.24
    2. Open lombok-1.18.24.jar
    3. Project Lombok installer -> Specify location... -> SpringToolSuite4.exe
    4. click Install/Update
